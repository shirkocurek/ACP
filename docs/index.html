Quick Install (Android)

  1. From this page, tap Download APK.
  2. On your device, approve install from your browser (one-time) and proceed.
  3. Open the app → allow notifications when prompted (recommended).

Tested on Android 11+; works on emulators and physical devices.

Why Android first?

Public, no-account installs are feasible on Android via APK. iOS distribution requires TestFlight or device registration (Apple Developer Program). See README for iOS options and next steps.

What’s inside

  • CRUD for Terms, Courses, Assessments (Objective/Performance)
  • Date & status pickers with validation
  • Local notifications (start/end), no server required
  • Instructor info & share notes via system share sheet

Verification & Integrity

Optional: compare the APK’s SHA-256 with the hash listed in the Release notes.

  • Windows (PowerShell): Get-FileHash .\AcademicCollegePlanner.apk -Algorithm SHA256
  • macOS/Linux: shasum -a 256 AcademicCollegePlanner.apk

Privacy & Permissions

The app stores data locally on your device using SQLite and only requests notification permission to schedule reminders you opt into. No accounts, analytics, or third-party trackers.

Hosting: GitHub Pages (static) + GitHub Releases (binaries). For questions, open a GitHub issue.
